Install SeedVR2 (nightly) for ComfyUI

macOS / Linux / WSL
Windows (Git Bash)
One-liner

cd ComfyUI/custom_nodes
git clone https://github.com/numz/ComfyUI-SeedVR2_VideoUpscaler.git
cd ComfyUI-SeedVR2_VideoUpscaler
git switch nightly
    

# Run in Git Bash from your ComfyUI folder:
cd ComfyUI/custom_nodes
git clone https://github.com/numz/ComfyUI-SeedVR2_VideoUpscaler.git
cd ComfyUI-SeedVR2_VideoUpscaler

cd ComfyUI/custom_nodes && git clone https://github.com/numz/ComfyUI-SeedVR2_VideoUpscaler.git && cd ComfyUI-SeedVR2_VideoUpscaler && git switch nightly
